#Personel Takip Sistemi
Bu proje, bir şirketteki personelleri yönetmek için geliştirilmiş bir **Spring Boot** uygulamasıdır.

### 🛠 Kullanılan Teknolojiler
* **Java 17 & Spring Boot 3**
* **Spring Data JPA** (Veritabanı yönetimi için)
* **MS SQL Server** (Veri saklama için)
* **Thymeleaf & Bootstrap** (Arayüz için)

### ⚙️ Kurulum ve Çalıştırma
1. SQL Server'da `PersonelDB` oluşturun.
2. `DB_USERNAME` ve `DB_PASSWORD` ortam değişkenlerini tanımlayın.
3. Projeyi çalıştırın ve `http://localhost:8080/api/employees` adresine gidin.
